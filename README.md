# Stock Screener

An AI-powered stock screening platform designed to assist investors in making better trading and investment decisions by combining quantitative screening, machine learning, and conversational AI.

## Overview

This project automates the stock screening process and provides intelligent insights through an AI assistant. The system periodically analyzes market data, generates screening results, and makes them available for both a Telegram Bot and a Web Dashboard.

## Features

* Automated stock screening
* AI-powered investment assistant
* Machine learning-based prediction
* Telegram Bot integration
* Web Dashboard
* Scheduled screening with GitHub Actions
* FastAPI backend
* Cloud deployment with Render and Vercel

## Project Status

🚧 This project is currently under active development.

Additional features, documentation, and deployment instructions will be added progressively.

## Tech Stack

* Python
* FastAPI
